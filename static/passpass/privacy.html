PassPass 개인정보처리방침

시행일: 2025년 1월 1일  |  최종 수정: 2026년 2월 24일

1. 개요

PassPass(이하 "이 확장프로그램")는 공공기관·금융 사이트의 OACX 간편인증 팝업에서 이름·생년월일·휴대폰번호를 자동으로 입력해주는 Chrome 확장프로그램입니다.

이 방침은 이 확장프로그램이 수집하는 정보의 종류, 이용 방법, 보관 방법 및 이용자의 권리를 안내합니다.

🔒 모든 개인정보는 이용자의 기기(브라우저)에만 저장됩니다. 개발자 서버 또는 외부 서버로 전송되지 않습니다.

2. 수집·저장하는 정보 로컬 저장

이 확장프로그램은 자동입력 기능을 위해 아래 정보를 이용자의 브라우저(chrome.storage.local)에 저장합니다.

항목 목적 저장 위치
이름 간편인증 폼 자동입력 기기 내 브라우저 로컬 저장소
생년월일 (8자리) 간편인증 폼 자동입력 기기 내 브라우저 로컬 저장소
휴대폰번호 간편인증 폼 자동입력 기기 내 브라우저 로컬 저장소
선호 인증기관 / 통신사 인증기관 자동 선택 기기 내 브라우저 로컬 저장소
비밀번호 해시값 (PIN) 확장프로그램 잠금 해제 인증 기기 내 브라우저 로컬 저장소
세션 인증 상태 브라우저 종료 전까지 잠금 유지 기기 내 브라우저 세션 저장소 (브라우저 종료 시 자동 삭제)

비밀번호(PIN)는 원문이 아닌 해시값으로만 저장되며, 원래 비밀번호는 어디에도 저장되지 않습니다.

3. 정보 이용 목적

저장된 정보는 오직 다음 목적으로만 사용됩니다.

  • 간편인증(OACX) 폼에 이름·생년월일·휴대폰번호 자동입력
  • 선호 인증기관 자동 선택
  • PIN을 통한 확장프로그램 잠금 및 잠금 해제

광고, 마케팅, 분석, 프로파일링 등 다른 어떠한 목적으로도 사용하지 않습니다.

4. 외부 전송 정보

이 확장프로그램은 기본적으로 외부로 정보를 전송하지 않습니다. 단, 이용자가 피드백 기능을 직접 사용하는 경우에 한해 아래 정보가 전송됩니다.

항목 전송 대상 조건
이용자가 직접 입력한 피드백 내용 개발자 텔레그램 (Telegram Bot API) 이용자가 "보내기" 버튼을 직접 클릭한 경우에만
피드백 전송 시점의 현재 페이지 URL 개발자 텔레그램 (Telegram Bot API) 이용자가 "보내기" 버튼을 직접 클릭한 경우에만

피드백 전송은 자동으로 이루어지지 않으며, 이용자가 명시적으로 전송 버튼을 눌러야만 발생합니다. 이름·생년월일·휴대폰번호 등 저장된 개인정보는 피드백 전송에 포함되지 않습니다.

5. 제3자 제공 및 위탁

이 확장프로그램은 이용자의 개인정보를 제3자에게 제공하거나 처리를 위탁하지 않습니다.

6. 보관 기간 및 파기

  • 저장된 개인정보는 이용자가 직접 초기화(데이터 삭제) 버튼을 누르거나 확장프로그램을 삭제하기 전까지 기기에 보관됩니다.
  • 세션 인증 상태는 브라우저 종료 시 자동으로 삭제됩니다.
  • 확장프로그램 제거 시 chrome.storage에 저장된 모든 데이터는 브라우저에 의해 자동 삭제됩니다.

7. 이용자 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람: 팝업 화면에서 저장된 정보 확인 가능
  • 수정: 팝업 → "정보 수정" 버튼으로 직접 수정 가능
  • 삭제: 팝업 → "초기화(데이터 삭제)" 버튼으로 모든 데이터 즉시 삭제
  • 확장프로그램 제거: Chrome 확장 관리 페이지에서 제거 시 모든 데이터 삭제

8. 보안

  • PIN(비밀번호)은 해시값으로만 저장되어 원문 복원이 불가합니다.
  • 개인정보는 chrome.storage.local에만 저장되며, 동기화(chrome.storage.sync)를 사용하지 않아 Google 서버에 업로드되지 않습니다.
  • 확장프로그램이 실행되는 웹페이지의 JavaScript 컨텍스트와 격리된 환경에서 동작합니다.

9. 만 14세 미만 어린이

이 확장프로그램은 만 14세 미만 어린이를 대상으로 하지 않습니다. 만 14세 미만 이용자의 개인정보를 의도적으로 수집하지 않습니다.

10. 개인정보처리방침 변경

방침이 변경될 경우 GitHub 저장소(github.com/seunghan91/passpass)와 이 페이지를 통해 공지합니다. 변경 전에 이 페이지에서 확인하시기 바랍니다.
