## 어떤 문제를 해결하나요?

홈택스, 정부24, 건강보험공단 등 공공기관 사이트에서 본인인증을 할 때마다
이름 · 생년월일 · 휴대폰번호를 반복 입력하는 불편함을 없애드립니다.

---

## 주요 기능

| 기능 | 지원 사이트 |
|------|------------|
| **OACX 간편인증 자동입력** | 홈택스, 정부24, 건강보험, 국민연금 등 50+ 사이트 |
| **인증기관 자동 선택** | PASS · 카카오 · 토스 · 국민인증서 · 네이버 · 신한 · 하나 · 우리 · NH · 삼성패스 |
| **KMC 본인인증 자동화** | kmcert.com 연동 사이트 전체 |
| **NICE 본인인증 자동화** | nice.checkplus.co.kr 연동 사이트 전체 |
| **토스 인증 자동입력** | auth.cert.toss.im |
| **PIN 잠금 보안** | 4자리 PIN으로 개인정보 보호 |

## 개인정보 보호

- 모든 정보는 **내 브라우저 로컬 스토리지에만 저장**됩니다
- 외부 서버로 개인정보를 전송하지 않습니다
- PIN 잠금으로 제3자 접근을 차단합니다

---

## 설치 방법

1. [Chrome 웹 스토어](https://chrome.google.com/webstore/detail/dpignhngmpbbnekagndefmlkoifpcemm)에서 **Chrome에 추가** 클릭
2. 확장프로그램 아이콘 클릭 → **PIN 설정**
3. 이름 · 생년월일 · 휴대폰번호 · 선호 인증기관 등록
4. 공공기관 사이트에서 간편인증 버튼 클릭 → **자동 입력 완료!**
