PassPass(이하 "확장프로그램")는 사용자의 개인정보를 소중히 여깁니다. 본 방침은 확장프로그램이 수집·저장·처리하는 정보와 그 방법을 설명합니다.

---

## 1. 수집하는 정보

확장프로그램은 사용자가 **직접 입력**한 아래 정보를 수집합니다.

| 항목 | 용도 |
|------|------|
| 이름 | 간편인증(OACX·KMC·NICE) 폼 자동입력, 다이렉트 보험 폼 자동입력 |
| 생년월일 | 간편인증 폼 자동입력, 다이렉트 보험 폼 자동입력 |
| 휴대폰번호 | 간편인증 폼 자동입력, 다이렉트 보험 폼 자동입력 |
| 선호 인증기관 | OACX 인증기관 자동 선택 |
| 통신사 / 인증방식 | KMC·NICE 인증 자동화 |
| PIN(4자리) 해시값 | 잠금 기능 보안 |
| 쿠팡 이메일 · 비밀번호 | 쿠팡 로그인 자동입력 (선택 입력) |
| 네이버 ID · 비밀번호 | 네이버 로그인 자동입력 (선택 입력) |
| 카카오 계정(이메일/ID) · 비밀번호 | 카카오 로그인 자동입력 (선택 입력) |

쿠팡·네이버·카카오 로그인 정보는 사용자가 직접 등록한 경우에만 저장되며, 등록하지 않으면 수집되지 않습니다.

---

## 2. 정보 저장 방식

- 모든 정보는 **사용자 기기의 브라우저 로컬 스토리지(`chrome.storage.local`)에만 저장**됩니다.
- 잠금 해제 세션 상태는 `chrome.storage.session`에 저장되며, **브라우저 종료 시 자동 삭제**됩니다.
- `chrome.storage.sync`는 사용하지 않으므로 **Google 서버에 동기화되지 않습니다**.
- PIN은 SHA-256 해시로 저장되며, 원본 값은 저장되지 않습니다.
- 쿠팡·네이버·카카오 비밀번호는 로컬 스토리지에 암호화 없이 저장됩니다. PIN 잠금으로 제3자 접근을 차단합니다.

---

## 3. 정보 전송 및 공유

- 확장프로그램은 수집한 모든 개인정보를 **어떠한 외부 서버로도 전송하지 않습니다**.
- 개발자를 포함한 제3자는 사용자의 개인정보에 접근할 수 없습니다.
- 사용자가 피드백 버튼을 직접 클릭할 때, **피드백 텍스트와 현재 페이지 URL만** Telegram API를 통해 전송됩니다. 이름·생년월일·휴대폰번호·비밀번호 등 개인정보는 포함되지 않습니다.

---

## 4. 정보 삭제

- 확장프로그램 팝업의 **"초기화 (데이터 삭제)"** 버튼을 누르면 저장된 모든 정보가 즉시 삭제됩니다.
- 쿠팡·네이버·카카오 로그인 정보는 설정 화면에서 개별 삭제도 가능합니다.
- Chrome에서 확장프로그램을 제거하면 로컬 스토리지에 저장된 모든 정보가 함께 삭제됩니다.

---

## 5. 권한 사용 목적

| 권한 | 사용 목적 |
|------|----------|
| `storage` | 인증 정보·로그인 정보·PIN 해시·세션 상태 로컬 저장 |
| `activeTab` | 간편인증·보험·로그인 폼이 열린 탭에서 자동입력 |
| `tabs` | 피드백 전송 시 현재 URL 첨부, PIN 입력 팝업 요청 처리 |
| 호스트 권한 | 공공기관·다이렉트 보험·쿠팡·네이버·카카오 등 지원 도메인에서 content script 실행 |

---

## 6. 보안

- PIN 잠금 기능으로 제3자의 접근을 방지합니다.
- 브라우저 종료 시 잠금 상태가 초기화됩니다.
- 확장프로그램 소스코드는 오픈소스로 공개되어 동작 방식을 직접 확인할 수 있습니다.
