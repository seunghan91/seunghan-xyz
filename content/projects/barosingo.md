## 바로신고

다산콜센터(120), 경찰청(112), 소방서(119)에 문자 기반 긴급 신고를 빠르고 간편하게 할 수 있는 앱입니다.

---

## 주요 기능

- **신고 대상 선택** — 다산콜센터, 경찰청, 소방서 중 선택
- **간편한 신고 작성** — 상황 설명 입력 + 사진 첨부
- **신고 이력 관리** — 접수완료, 처리중 등 상태 실시간 확인
- **알림 기능** — 신고 접수 및 처리 완료 푸시 알림

---

## 기술 스택

| 영역 | 기술 |
|------|------|
| Backend | Rails 8 + Turbo |
| iOS | Hotwire Native 1.2.2 (Swift, XcodeGen) |
| Android | Hotwire Native 1.2.5 (Kotlin) |
| 호스팅 | Render |
